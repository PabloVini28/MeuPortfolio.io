Minha história

Comecei na programação em 2021 durante meu ensino médio, percebi que as linguagens que eu utilizava para criar mini-games e resolver problemas matemáticos iam muito além disso. Ao ingressar no curso de Eng. de Computação, em 2022, na UFC Quixadá, meu grande sonho têm sido me tornar um grande programador e, através das minhas aplicações, ter o poder de mudar a vida das pessoas ao meu redor e, desse modo, tornar o mundo um lugar melhor para viver.

Atualmente, sou entusiasta dos estudos envolvendo programação Full-Stack, tendo em vista os conceitos de HTML5,CSS e JavaScript com ênfase em FrameWorks como o React e o VueJS no que tange a área do Front-End. Por outro lado, estou aprofundando meus estudos em Java e Python para desenvolvimento Back-End, utilizando Spring e Flask como FrameWorks de estudo.

Além disso, estou me aprofundando em estudos envolvendo Data Science, visando me tornar um cientista de dados eficiente e proativo. O estudo de probabilidade e estatística, bem como a aplicação de Machine Learning e o uso de Redes Neurais têm sido meu maior foco.

Minha carreira como Dev

Como foi dito anteriormente, comecei a usar a linguagem Python para desenvolver mini-games simples como por exemplo, um Pedra,Papel ou Tesoura ou um jogo de sorteio de número aleatórios. Além disso, era comum para mim usar o Python para resolver pequenos problemas matemáticos, creio que todos aqueles que seguem ou sonham em seguir uma carreira como Dev já se encantaram ao fazer sua primeira calculadora

Primeiros passos...

Ao ingressar no curso de Eng.de Computação tive meu primeiro contato com a linguagem C, uma linguagem estruturada de baixo nível, com uma sintaxe diferente e bem mais próxima do Hardware do que o Python. Na cadeira de FUP, desenvolvi ao lado de mais 2 colegas um sistema feito inteiramente com linguagem C que tinha como devido fim, simular uma locadora de filmes Trabalho de FUP.

Logo em seguida, tendo em vista o total preparo para montar um código organizado, estruturado e documentado, segui para a cadeira de Estrutura de Dados onde aprendi a manipular conjuntos de dados através da linguagem C++. A cadeira abordou um conteúdo que envolve vários tipos de estruturas como a lista, a fila,a pilha e árvore binária. Desenvolvi 2 projetos ao lado de um colega para esta cadeira Trabalho de ED.

Atualmente

Graduando de Eng.de Computação, possuo habilidades com linguagens de baixo nível como Assembly-x86 e ARM, desenvolvo aplicações BareMetal e programo Sistemas Embarcados. Além disso, possuo experiência com C#,Python,Java,JavaScript e Banco de Dados MySQL. Pretendo até o final da graduação me tornar um programador Full-Stack e um Cientista de Dados para atender as demandas do mercado de trabalho e, além disso, ser um programador de Sistemas Embarcados Sênior
